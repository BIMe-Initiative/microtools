# BIMei Chatbot Active Files - Product Overview

## Project Purpose
This project contains the active implementation files for the BIMei (BIM Excellence Initiative) Assistant chatbot system. It provides an intelligent conversational interface that helps users navigate and query the BIMei knowledge base, offering expertise in BIM competencies, tools, and methodologies.

## Key Features and Capabilities

### Core Functionality
- **Intelligent Query Routing**: Automatically routes relationship and graph-based queries to specialized handlers
- **Knowledge Graph Integration**: Connects to Neo4j database for complex relationship queries and path analysis
- **Multi-modal Response System**: Supports both default (100-150 words) and expert mode (up to 300 words) responses
- **Custom UI Interface**: Provides tailored web interface for BIMei-specific interactions
- **Vertex AI Integration**: Leverages Google Cloud's Vertex AI for advanced language processing

### Specialized Capabilities
- **Relationship Analysis**: Handles queries about connections, dependencies, and associations between BIM concepts
- **Path Discovery**: Finds shortest paths and network traversals within the knowledge graph
- **Aggregation Queries**: Processes "how many", "top N", and clustering requests
- **Summarization**: Provides intelligent summaries of pages and documents
- **Translation Support**: Multi-language translation capabilities

## Target Users and Use Cases

### Primary Users
- **BIM Professionals**: Seeking guidance on competencies and methodologies
- **Researchers**: Exploring relationships between BIM concepts and tools
- **Students**: Learning about BIM excellence frameworks and standards
- **Consultants**: Accessing structured knowledge for client recommendations

### Key Use Cases
- Competency assessment and maturity evaluation
- Finding relationships between BIM tools and methodologies
- Discovering implementation pathways for BIM initiatives
- Accessing structured knowledge with evidence-based responses
- Expert-level technical queries with detailed semantic analysis

## Value Proposition
The BIMei Assistant transforms complex BIM knowledge into accessible, contextual responses while maintaining academic rigor and providing traceable evidence sources. It bridges the gap between comprehensive BIM expertise and practical, actionable guidance for professionals at all levels.
