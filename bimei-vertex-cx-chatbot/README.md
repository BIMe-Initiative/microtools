# BIMei Vertex CX Chatbot

Intelligent conversational interface for BIM Excellence Initiative knowledge base with Google Cloud Vertex AI integration.

## Features
- Vertex AI integration for natural language processing
- Neo4j graph database for relationship queries
- WordPress-compatible UI with glassmorphism design
- Structured JSON responses with evidence tracking

## Quick Start
1. Configure Google Cloud credentials
2. Set up Neo4j database connection
3. Deploy to Google Cloud Run or Functions
4. Integrate UI with WordPress

## Architecture
- **Frontend**: WordPress-safe HTML/CSS/JS
- **Backend**: Node.js with Vertex AI proxy
- **Database**: Neo4j graph database
- **AI**: Google Cloud Vertex AI
